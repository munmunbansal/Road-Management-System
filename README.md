# Road Management System & Highway Analytics Dashboard

## Overview

A data-driven infrastructure management platform developed using National Highway datasets from Government of India sources.

## Features

- State-wise highway project delay analysis
- Delay percentage and completion tracking
- Power BI dashboards for infrastructure monitoring
- Citizen road issue reporting system (In Development)
- Engineer issue tracking and resolution portal (In Development)

## Technologies

- Power BI
- Excel
- React.js
- Node.js
- Express.js
- MongoDB
